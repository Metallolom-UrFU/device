Простенький веб-сервер на FastAPI для взаимодействия:
- Пользователя с устройством
- Устройства с удалённым сервером.

Основные страницы:
- Меню
- Получение заказа по QR-коду в приложении
- Возврат книги по QR-коду на ней
- Ссылка для скачивания приложения (пока в качестве заглушки ведёт на наше хранилище)
